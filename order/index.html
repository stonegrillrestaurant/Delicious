How to Order

  1. Go to stonegrillresto.net/order (mobile or desktop).
  2. Select a category and tap Add beside each dish.
  3. Review Your Order; adjust quantities or remove items.
  4. Fill in:
    • Full Name & Mobile (+63 auto)
    • Order Type (Dine-in / Take-out)
      • If Dine-in → enter Number of Persons.
    • Date & Time for dine-in/pick-up
    • Special Requests (optional)
  5. Click Submit Order — you’ll see a confirmation toast.
  6. We’ll message you for payment instructions. Orders are processed after payment.

Tip: Keep your phone nearby in case we need to confirm details.
